## Задача 5: Полный гайд по аудиту и запуску рекламы (12 февраля, 17:40)

**Что сделано:**
- Проанализированы существующие документы аудита (audit_report_gulder_kz.md, strategy_relaunch_gulder.md)
- Создан комплексный гайд: `guide_full_ads_launch.md`

**Ключевые выводы:**

### Почему реклама перестала быть прибыльной (5 причин):
1. Стратегия «Макс. кликов» вместо «Макс. конверсий»
2. Гео-настройка «Присутствие ИЛИ интерес» → слив 30-40% бюджета
3. Массовые отклонения объявлений
4. Дорогая Астана (CPA $28 vs $15 среднее)
5. Нет оптимизации за 10 лет

### Полнота аудита:
- Покрыто ~60% (кампании, ключевые, запросы, теги, объявления)
- Не покрыто: аудитории, расширения, конкуренты, устройства, время, атрибуция

### План запуска:
- Минимальный старт: $75-100/день (Бренд + Диаспора)
- Ожидаемый CPA: $5-7 → прибыль 2,9-4,6 млн тг/мес

**Победы:**
- Полная картина по аккаунту gulder.kz сформирована
- Конкретный план запуска с бюджетами и ключевыми словами
- Стратегия из 3 фаз: старт → масштабирование → оптимизация

**Проблемы:**
- Нужно переименовать бренд с gulder на Henry Bonnar во всех кампаниях
- Нужно создать новые объявления с нуля (старые отклонены Google)
- Сайт требует редизайна по ТЗ перед запуском рекламы
